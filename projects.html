Overview

The following projects (AI & IPD) have been developed as part of final year projects at Macquarie University.


PROJECT_AI: This is basically a conversation agent design to perform customer service with the help of machine learning Artificial Intelligence. The main objective of this agent is to deliver timetable and track work information to cityRail customers.


PROJECT_IPD: In this project a game called Iterative Prisoner's dilemma (IPD) has been implemented. This game is mainly played among two players (as prisoners) for the best possible outcome for serving 'LESS' prison terms in relation to their 'same crime'. In this game each player's move is designed by using re-production rules (cross-over & mutation) of 'Genetic Algorithm'.


FEEDBACK CONTROL SYSTEM: Feedback control system comprises a database that stores messages from users who provides feedback in this web application for the above projects. Note: Admin can view all messages with the following credentials: username: 'admin', password: 'a'
