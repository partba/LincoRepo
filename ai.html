Conversational Agent

Overview:

This is a prototype of a text based conversational agent. In this dialogue system, AI is capable of conversing with a human in ordinary language and provide information on timetable and track-work for the following stations and lines.

Version: 01

Main Features:

  • Timetable info for Sydney Northern Stations:
    • Hornsby
    • Epping
    • Eastwood
    • Central and so on...
      •
  • Planned Track-work Info for:
    • Northern Line
    • Southern Line
    • Western Line
    • Bankstown Line

    • Development Specification: This prototype is developed, using Python3.6 with embedded SQLite database
